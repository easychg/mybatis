public static void main(String[] args) throws IOException {
        String resource="mybatis-config.xml";
        InputStream inputStream=Resources.getResourceAsStream(resource);
        SqlSessionFactory sqlSessionFactory=new SqlSessionFactoryBuilder().build(inputStream);
        SqlSession session=sqlSessionFactory.openSession();
         
        List<Integer> ids = new ArrayList();
        ids.add(1);
        ids.add(6);
        ids.add(2);
         
        listByIDs(session, ids);
         
        session.commit();
        session.close();
    }
 
//遍历ids的方法
private static void listByIDs(SqlSession session,List<Integer> ids) {
        List<Product> ps=session.selectList("listProductIter", ids);
        for (Product p : ps) {
            System.out.println(p);
        }
    }

lxt 答案时间：2017-09-19
这个很好理解，就是把类似如下sql select * from product_ where id in ('1','3','5') 中in 后面的 ('1','3','5')拆成open="(" separator="," close=")"这种写法，open='('代表in后面的开始的括号，close=')'代表结束的括号，而seperator=','则代表括号里面数据的分隔符，那么具体的1，3，5则用占位符#{item}进行替换
